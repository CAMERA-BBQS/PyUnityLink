# PyUnityLink

## About
A Python-based central orchestration service that coordinates control signals and data flow between BCI2000 and an iPad Unity client.
It is intended for educational and demonstration purposes and does not include clinical and medical data.

This repository is part of the CAMERA Platform.
See the main guide here: [CAMERA-Platform_Overview](https://github.com/CAMERA-BBQS/CAMERA-Platform_Overview)
